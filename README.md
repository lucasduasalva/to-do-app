# To-Do List

Projeto de lista de tarefas desenvolvido como atividade acadêmica, com foco na prática de HTML, CSS e JavaScript.

## Funcionalidades

- Adicionar novas tarefas  
- Definir prioridade (Alta, Média, Baixa)  
- Exibir tarefas dinamicamente na tela  
- Marcar tarefas como concluídas  
- Aplicar destaque visual de acordo com a prioridade  

## Tecnologias utilizadas

- HTML5  
- CSS3  
- JavaScript  

## Como utilizar

1. Digite uma tarefa no campo de entrada  
2. Selecione a prioridade desejada  
3. Clique no botão "Adicionar"  
4. A tarefa será exibida na lista  
5. Clique em "Concluir" para marcar como finalizada  

## Estrutura do projeto

- index.html → Contém toda a estrutura, estilo e lógica do projeto  

## Objetivo do projeto

Este projeto foi desenvolvido com o objetivo de aplicar conceitos fundamentais de desenvolvimento web, incluindo:

- Manipulação do DOM  
- Eventos em JavaScript  
- Estruturação semântica com HTML  
- Estilização com CSS  
- Organização de código
